
quando você compila um código em uma IDe, nada mais que a IDE executa o seu arquivo pelo prompt em alguma pasta,
se você observar o OUTPUT equanto executa verá os comandos de execução do seu arquivo.
no linux, por exemplo, a IDE executa "./OBS:.c" em alguma pasta, no meu salva na pasta OUTPUT
então, quando você executar pela IDE o arquivo será gerada na pasta que você está,
se no terminal você estiver em outro local, exemplo: C:/teste/user/manipular_arquivos -> o arquivo.txt irá para lá


vou criar 2 scripts para exemplificar isso:

 - se você usa Windows execute o .bat
 - se você usar Linux exute o .sh 

ó clicar em editar neles para ver o código de execução
